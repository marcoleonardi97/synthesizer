Lines from Grid objectsÂ¶

Grids that have been post-processed through a photoionisation code (e.g. Cloudy) contain information on emission lines. These can be loaded like regular grids, but there are a number of additional methods for working with lines, as demonstrated in these examples.

[1]:

import matplotlib.pyplot as plt
import numpy as np
import synthesizer.line_ratios as line_ratios
from synthesizer.grid import Grid
from synthesizer.line import (
    get_diagram_labels,
)

Letâ€™s first introduce the line_ratios module. This contains a set of useful definitions.

[2]:

# the ID of H-alpha
print(line_ratios.Ha)

# the available in-built line ratios ...
print(line_ratios.available_ratios)

# ... and diagrams.
print(line_ratios.available_diagrams)

grid_dir = "../../../tests/test_grid"
